How It Works

This uses the GitHub REST API to fetch publicly available data like followers, repositories, and commits.

How It Is Graded

A total of 40 points is awarded. Each section gives 10 points to the winner. The follower and following sections are split — 5 points each.

Projects / Repositories

A project is only counted if it meets all 3 criteria:
• It's public
• It has a README
• It has at least 5 commits
